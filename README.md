# Streamlit Projects

Repositorio que contiene aplicaciones interactivas desarrolladas con Streamlit orientadas a análisis de datos, optimización y simulación.

Cada subcarpeta incluye un proyecto independiente con su propia documentación y archivos necesarios para su ejecución.

## Contenido del repositorio

- league-genetic-optimizer/
  Aplicación que utiliza un algoritmo genético para calcular la jornada ideal de liga.

- NY-simulator-app/
  Aplicación de simulación basada en modelos de Machine Learning para predicción interactiva de precio de vivienda.

## Tecnologías utilizadas

- Python
- Streamlit
- NumPy
- Pandas
- Matplotlib
- Scikit-learn
- LightGBM / XGBoost (según proyecto)
